# Live Currency Converter

## Overview
The **Live Currency Converter** is a web-based application that allows users to convert one currency into another using real-time exchange rates. The application provides a simple and user-friendly interface where users can enter an amount, select currencies, and instantly view the converted value.

## Features
- Real-time currency conversion
- Multiple currency support (USD, INR, EUR, GBP, JPY)
- Clean and responsive user interface
- Error handling for invalid inputs
- Fast API-based exchange rate retrieval

## Technologies Used
- **HTML** – Structure of the web page  
- **CSS** – Styling and layout design  
- **JavaScript** – Application logic and currency conversion  
- **Fetch API** – Retrieving live exchange rate data  
- **ExchangeRate API** – Source of real-time currency data

## How It Works
1. The user enters the amount to convert.
2. The user selects the source currency and target currency.
3. The application sends a request to the **Exchange Rate API**.
4. The API returns the latest exchange rate.
5. The converted amount is calculated and displayed to the user.

## Installation and Usage
1. Download or clone the repository.
2. Open the project folder.
3. Run the `index.html` file in any web browser.
4. Enter the amount and select currencies to perform the conversion.
